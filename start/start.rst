开箱教程
===============

开箱提醒
-----------

此章节主要让各位用户和小白能够使用最少的步骤来点亮Sipeed M1。少量甚至无需配置，让你快速上手。通过串口输出，摄像头输出，人脸识别，点亮pwm灯，让你最快
验证开发板。更加详细的说明，将在后文进行介绍。

如果您和笔者一样是个小白，希望能给您的学习提供一点小小的帮助。

如果您是大佬，并且熟悉开发流程，快速浏览即可 :)

.. note:: 该上手教程适用于Sipeed M1/MW dock套装


**开发环境及工具**:

- ubuntu18.04/window

- TypeC 数据线/转接头

- Sipeed M1/MW dock板及相关配件

如果您够买的是Sipeed M1/MW dock套装，开箱之后将包含以下物品  

OV2640摄像头，ST7789显示屏，M1/MW dock板，TypeC 转接口
